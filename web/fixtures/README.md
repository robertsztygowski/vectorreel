# Fixtures

Byte-identical mirror of `tests/fixtures/output/` — the canonical Phase-2.5 output-contract
fixtures (see the README there). Generated by `scripts/sync-fixtures.mjs`. **Do not hand-edit.**
Re-run `npm run sync-fixtures` after the canonical fixtures change, then commit the result —
`tests/` is invisible to the Cloud Run source-deploy build context, so the app reads these
mirrored copies at request time, not the originals. `npm run sync-fixtures -- --check` verifies
the mirror (part of the contract test suite).
